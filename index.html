Rules for Variable Names

must begin with a letter/$/_

can contain letter/number/$/_

must not contain -/.

can not use keywords/reserved words

use camelCasing or _ for longer variable names

! variables are case sensitive

Arrays

colors = ['red', 'green', 'blue']; - array literal

colors = new Array('red', 'green', 'blue'); - array constructor

colors[0]; - access a value in an array literal

colors.item(0); - access a value in an array constructor

colors[0] = 'white'; - change a value in an array literal

colors.item(0) = 'white'; - change a value in an array constructor

colors.length - access the number of items in an array

Expressions

Functions, Methods & Objects

function doSomething() {...} - function declaration

doSomething(); - calling a function

function doStuff(paramOne, paramTwo) {...} - fn declaration with the parameters

doStuff(argOne, argTwo); - fn call with the arguments to be used by the function

! arguments can be passed as values or as variables

Return

Get a single value out of the function:

function addNumbers(numberOne, numberTwo) {
var newNumber = numberOne + numberTwo;
return newNumber;
}
var totalPrice = addNumbers(12, 24);

Output:

Get an array of values out of the function:

function renovation(floorX, floorY, height) {
var floorArea = floorX * floorY;
var wallArea = (floorX * 2 + floorY * 2) * height;
var roomAreas = [floorArea, wallArea];
return roomAreas;
}
var myWalls = renovation(4, 5, 3)[1];

Output:

Objects

Inside the object:

  • Variables become properties
  • Functions become methods

Creating an object: literal notation:

var item = {
name: 'PS5',
stock: 1000,
ordered: 999,
available: function() {
return this.stock - this.ordered;
}
};

Creating an object: constructor notation:

var item = new Object();
item.name = 'PS5';
item.stock = 1000;
item.ordered = 999;
item.available = function() {
return this.stock - this.ordered;
};

Accessing an object: dot notation:

var itemName = item.name;
var availability = item.available();

Output:

Updating an object: dot notation:

item.stock = 9000;
item.ordered = 6969;

Accessing an object: bracket syntax:

var itemName = item['name'];

Updating an object: bracket syntax:

item['name'] = 'Xbox';

! you can not access/update methods using bracket syntax

Creating many objects: constructor notation (function):

function OledTv(name, size, price) {
this.name = name;
this.size = size;
this.price = price;
this.pricePerInch = function() {
return this.price / this.size;
};
}

! it is common to name the constructor function capitalized

Creating & accessing instances of the object:

var lgC148 = new OledTv('LG C1', 48, 989);
var lgC155 = new OledTv('LG C1', 55, 1240);
var lg48Pri = lgC148.price;
var lg48Ppi = lgC148.pricePerInch();

Output:

Removing and adding properties and methods:

delete someObject.someProperty;
someObject.someNewProperty = 'Yay! A New Property';

Objects can also have arrays as property values:

someObject.someNewProp = [16, 32, 64, 128, 256];
someObject.someNewProp[2]; //access

Loops

! In general, you should use a for loop when you know how many times the loop should run. If you want the loop to break based on a condition other than the number of times it runs, you should use a while loop.

for loop (counter loop):

for (var i = 0; i < 10; i++) {
run this code;
}

(var i = 0; i < 10; i++) - condition (counter)

var i = 0; - initialization

i < 10; - condition

i++ - update (i = i + 1)

while loop (condition loop):

while (condition) { //condition = true
run this code;
} (code will run as long as the condition is true)

do...while loop:

do {
run this code;
} while (condition); (code will run at least once even if the condition is not met)
